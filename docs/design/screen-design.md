## 10. アクセシビリティ

| 項目 | 対応 |
|------|------|
| コントラスト比 | 4.5:1 以上 (WCAG 2.1 Level A) |
| フォーカス表示 | キーボード操作時のフォーカスリング |
| aria-label | 全インタラクティブ要素に付与 |
| alt テキスト | 全画像に適切なalt |
| セマンティクス | heading階層の適切な使用 |
| フォーム | label/fieldset/legend の適切な使用 |
